# Movie Searcher

## What has to be done

Create a web app to do movie search.

Use the following api:
- https://www.omdbapi.com/

### Requirements:
✅ Show an input and button in order to perform the search.
- List all movies found and show its title, year of publication and poster.
- Show movies in a **responsive** grid.

### First iteration:
- Don't let the user perform the same search twice in a row.
- Make it so the search is done automatically while writing.
- Don't do the search continuously while writing (debounce).
